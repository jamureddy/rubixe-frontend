## Run Locally

Clone the project

```bash
  git clone https://github.com/jamureddy/rubixe-frontend
```

Go to the project directory

```bash
  cd rubixe-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
